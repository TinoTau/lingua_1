# iOS 多会话 UI 草图（SwiftUI 结构）

版本：v1.0

本文件展示多会话架构的 UI 草图，可交给设计或开发团队。

---

# 1. 顶层导航结构

```
+--------------------------+
| SessionsView (主界面)    |
+--------------------------+
            |
            +-- 点击进入某个会话
            v
+--------------------------+
| SessionView (单会话页面) |
+--------------------------+
```

---

# 2. SessionsView（会话列表）草图

```
+------------------------------------+
|   实时翻译会话                     |
+------------------------------------+
| [ + 新建会话 ]                     |
|------------------------------------|
| 会话 A：中文 → 英文               |
|   最近一句：你好                  |
|------------------------------------|
| 会话 B：英文 → 西班牙语           |
|   最近一句：How are you           |
|------------------------------------|
| 会话 C：中文 → 日语               |
|   最近一句：谢谢                  |
+------------------------------------+
```

功能：
- 新建会话
- 删除会话（左滑）
- 点击进入会话

---

# 3. SessionView（单会话页面）草图

```
+------------------------------------+
| < 返回                             |
| 中文 → 英文                        |
+------------------------------------+
|  你：今天天气不错                 |
|  AI：The weather is nice today.   |
|------------------------------------|
|  你：我们去吃饭吧                 |
|  AI：Let's go eat.                |
|------------------------------------|
|              ...                   |
+------------------------------------+
|   [ 开始翻译 ] 或 [ 停止翻译 ]      |
+------------------------------------+
```

---

# 4. DebugOverlay 草图

```
+----------------------------+
|   DEBUG PANEL              |
+----------------------------+
| WS: connected              |
| Upload: 32 KB/s            |
| Download: 24 KB/s          |
| RTT: 152 ms                |
| Session: sess-123          |
| Seq: 421                  |
| Last error: none           |
+----------------------------+
```

触发方式：
- 三指双击屏幕
- 或摇一摇（UITouch / motion event）

---

# 5. 多会话切换动画

```
SessionView ----(Back)-----> SessionsView ----> Another SessionView
```

开发参考：
- 使用 `NavigationStack`
- 每个会话绑定一个 SessionViewModel
