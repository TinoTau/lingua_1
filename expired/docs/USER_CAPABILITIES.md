## 🎯 核心功能

### 1. 实时语音翻译 🌐

**功能描述**:  
用户可以通过 Web 浏览器进行实时语音翻译，系统会将您说的话翻译成目标语言并播放出来。

**使用场景**:
- 跨语言对话（如中文 ↔ 英文）
- 国际会议实时翻译
- 语言学习辅助
- 无障碍交流

**工作流程**:
1. 打开 Web 客户端
2. 选择源语言和目标语言
3. 点击开始，开始说话
4. 系统实时识别您的语音
5. 自动翻译并播放翻译结果

---

### 2. 半双工交互模式 🎤🔊

**功能描述**:  
系统采用半双工模式，确保清晰的交互体验。

**特点**:
- ✅ **说话时**: 麦克风开启，系统正在识别您的语音
- ✅ **播放时**: 麦克风自动关闭，避免回声干扰
- ✅ **自动切换**: 系统自动在"输入"和"播放"模式间切换
- ✅ **节奏控制**: 您可以通过 Send 按钮主动结束当前发言

**优势**:
- 不会出现声音回录问题
- 翻译播放清晰，不受干扰
- 交互节奏自然流畅

---

### 3. 实时语音识别字幕 📝

**功能描述**:  
在您说话时，系统会实时显示语音识别结果，让您确认识别是否正确。

**特点**:
- ✅ **实时显示**: 边说边显示识别文字
- ✅ **即时反馈**: 立即知道系统是否正确识别
- ✅ **错误修正**: 如果识别有误，可以重新说话

**使用体验**:
- 看到识别结果，确认准确性
- 提高翻译质量（识别准确 = 翻译准确）
- 增强用户信心

---

### 4. 自动语种识别 🔍

**功能描述**:  
系统可以自动检测您说的语言，无需手动选择。

**特点**:
- ✅ **智能检测**: 自动识别您说的语言
- ✅ **多语言支持**: 支持多种语言的自动检测
- ✅ **双向模式**: 支持双向自动翻译（A ↔ B）

**使用场景**:
- 不确定对方语言时
- 多语言混合对话
- 简化操作流程

**当前支持**:
- 自动检测源语言
- 双向模式（自动判断翻译方向）

---

### 5. 上下文感知翻译 🧠

**功能描述**:  
系统会记住您之前说的话，让翻译更连贯、更自然。

**特点**:
- ✅ **上下文拼接**: 将连续的话语组合在一起
- ✅ **语义连贯**: 翻译结果更自然、更准确
- ✅ **自动分组**: 系统自动判断哪些话属于同一话题

**使用体验**:
- 说多句话时，翻译更连贯
- 避免短句切分导致的语义割裂
- 翻译结果更符合自然语言习惯

**示例**:
```
您说: "我想订一张机票"
系统: "I want to book a ticket"

您接着说: "明天去奥克兰的"
系统: "to Auckland tomorrow" （理解上下文，翻译更自然）
```

---

### 6. 功能模块选择 ⚙️

**功能描述**:  
用户可以根据需要选择启用哪些高级功能。

**可选功能**:
- 🎭 **情感检测**: 识别语音中的情感
- 🎨 **音色识别/生成**: 识别或生成特定音色
- ⚡ **语速识别/控制**: 识别或控制语速
- 👤 **说话人识别**: 识别不同的说话人
- 🎯 **个性化适配**: 根据用户习惯个性化翻译

**使用方式**:
- 在 Web 客户端界面选择需要的功能
- 系统会根据您的选择匹配相应的节点
- 未选择的功能不会启用，节省资源

---

**当前能力**:

## 📱 使用方式

### Web 客户端

1. **打开浏览器**: 访问 Web 客户端地址
2. **选择语言**: 选择源语言和目标语言（或使用自动检测）
3. **开始对话**: 点击开始按钮，开始说话
4. **查看字幕**: 实时查看语音识别结果
5. **控制节奏**: 使用 Send 按钮结束当前发言
6. **收听翻译**: 系统自动播放翻译结果

### 交互流程

```
[准备就绪] 
  ↓ 开始说话
[正在录音] → 实时显示识别字幕
  ↓ 点击 Send 或自动结束
[等待结果] 
  ↓ 收到翻译
[播放翻译] → 麦克风自动关闭
  ↓ 播放完成
[准备就绪] → 可以继续说话
```
