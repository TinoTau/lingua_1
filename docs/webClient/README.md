## 关键决策

根据 v3 方案，以下为关键决策：

1. **跳过播放按钮不是必须，也不做 v1 功能**
2. **ASR 字幕是必须功能**
3. **Utterance Group & 上下文拼接为最高优先级**
4. **静音阈值不可由用户配置（避免复杂度）**
5. **iOS 相关内容从文档中移除或降级为未来考虑**
