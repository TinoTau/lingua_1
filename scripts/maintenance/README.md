# 维护脚本

本目录包含系统维护相关的脚本工具。

## 脚本分类

### 日志清理
- `clear_logs_simple.ps1` - 简单日志清理脚本
- 清理系统运行产生的日志文件

### 配置修复
- `fix_port_config.ps1` - 端口配置修复脚本
- 修复端口配置相关问题

### 服务验证
- `verify_and_fix_service_json.ps1` - 服务配置验证和修复脚本
- 验证并修复服务配置JSON文件

## 使用说明

### 清理日志

```powershell
# 清理所有日志
.\clear_logs_simple.ps1
```

### 修复端口配置

```powershell
# 修复端口配置问题
.\fix_port_config.ps1
```

### 验证服务配置

```powershell
# 验证并修复服务配置
.\verify_and_fix_service_json.ps1
```

## 注意事项

1. **日志清理**
   - 清理前会备份重要日志
   - 建议定期执行以节省磁盘空间

2. **配置修复**
   - 修复前会自动备份原配置
   - 执行后需要重启相关服务

3. **服务验证**
   - 会检查所有服务配置的完整性
   - 自动修复常见配置错误

## 定期维护建议

- **每日**: 清理临时日志
- **每周**: 验证服务配置
- **每月**: 全面系统检查
