# ASR→FW Detector→Aggregation→NMT 主链继续冻结方案

版本：V1.0  
日期：2026-05-30  
状态：继续冻结实施方案  
适用项目：Lingua_1 / electron-node / FW-only 主链  
冻结目标：完成 `segmentForJobResult` 单真值源统一后，冻结 ASR 后处理至 NMT 前主链控制流

---

# 1. 执行摘要

当前 FW Detector 架构已经基本稳定：

```text
ASR
→ FW_SPAN_DETECTOR
→ AGGREGATION
→ DEDUP
→ TRANSLATION
```

已确认：

- `LEXICON_RECALL` 不再进入 FW 主链
- `SENTENCE_REPAIR` 不再进入 FW 主链
- CTC / Sherpa / n-best / rerun / secondary decode 不进入默认主链
- FW Detector 分层为 Detect → Recall → Decision → Apply
- KenLM 仅作为 weak veto
- 5015 / 5016 / 5017 默认关闭

当前唯一影响主链冻结的问题是：

```text
repairedText
vs
segmentForJobResult
```

存在双真值源。

本方案目标：

```text
以 segmentForJobResult 作为节点端 ASR 后、NMT 前唯一业务文本源
```

完成该统一后，正式冻结：

```text
ASR → FW Detector → Aggregation → NMT
```

的主链控制流。



# 4. 数据结构样例

## 4.1 PipelineContext

```ts
export interface PipelineContext {
  jobId: string;
  turnId?: string;
  sourceLang?: string;
  targetLang?: string;

  /**
   * ASR immutable raw text.
   * Only ASR step writes this field.
   * No repair / aggregation / translation step may overwrite it.
   */
  rawAsrText?: string;

  /**
   * ASR diagnostic text.
   * May represent concatenated ASR batch view.
   * Not the business source of truth.
   */
  asrText?: string;

  /**
   * Single source of truth after ASR.
   * All business steps read/write this field.
   */
  segmentForJobResult?: string;

  shouldDeferTranslation?: boolean;
  shouldAllowTranslation?: boolean;

  /**
   * Segment write lock.
   * True means a repair step has already produced authoritative segment text.
   */
  asrRepairApplied?: boolean;

  extra?: Record<string, unknown>;
}
```

---

## 4.2 Turn Buffer

```ts
interface TurnBuffer {
  turnId: string;
  sourceLang: string;
  targetLang: string;

  committedSegments: Array<{
    jobId: string;
    text: string;
    source: "segmentForJobResult";
    startMs?: number;
    endMs?: number;
  }>;

  accumulatedText: string;
}
```

---

## 4.3 Translation Input

```ts
interface TranslationInput {
  text: string;
  source: "segmentForJobResult";
  sourceLang: string;
  targetLang: string;
}
```

# 5. 接口冻结

## 5.1 ASR Step 输出契约

ASR Step 输出：

```ts
ctx.rawAsrText = asrResult.text;
ctx.asrText = asrResult.text;
ctx.segmentForJobResult = asrResult.text;
```

多 batch 场景：

```ts
ctx.asrText = concatBatchTexts(batchTexts);
ctx.rawAsrText = firstRawText;
ctx.segmentForJobResult = currentBusinessText;
```

约束：

- `rawAsrText` 仅 ASR Step 首次赋值
- `segmentForJobResult` 是后续主链唯一输入
- 不再写 `repairedText`

---

## 5.2 FW Detector 输入输出契约

输入：

```ts
const detectInput = ctx.rawAsrText;
```

输出：

```ts
ctx.segmentForJobResult = applyFwSpanReplacements(...);
ctx.asrRepairApplied = true;
```

约束：

- Detect 输入固定为 `rawAsrText`
- Apply 输出固定写 `segmentForJobResult`
- 不写 `repairedText`
- 不调用旧 Recover step
- KenLM 只做 weak veto

---

## 5.3 Aggregation 契约

### non-finalize

```ts
appendTurnSegment(ctx.segmentForJobResult);

ctx.shouldDeferTranslation = true;
ctx.shouldAllowTranslation = false;
```

约束：

- 不清空 `segmentForJobResult`
- 不写 `repairedText`
- 不触发 Translation

### finalize

```ts
ctx.segmentForJobResult = mergeTurnSegments(...);
ctx.shouldDeferTranslation = false;
ctx.shouldAllowTranslation = true;
```

约束：

- finalize 后 `segmentForJobResult` 必须是完整 turn 文本
- NMT 输入必须使用该完整 turn 文本
- Result `text_asr` 必须等于完整 turn 文本

---

## 5.4 Translation 契约

```ts
export function getTextForTranslation(ctx: PipelineContext): TranslationInput {
  const text = ctx.segmentForJobResult?.trim();

  if (!text) {
    throw new Error("Translation input missing: segmentForJobResult is empty");
  }

  return {
    text,
    source: "segmentForJobResult",
    sourceLang: ctx.sourceLang!,
    targetLang: ctx.targetLang!
  };
}
```

约束：

- 不读取 `repairedText`
- 不优先读取 `asrText`
- 不新增 `finalText`
- 不新增 `normalizedText`
- 不新增第三业务文本字段

---

## 5.5 Result Builder 契约

```ts
const textAsr = ctx.segmentForJobResult?.trim() || "";
```

输出：

```ts
result.text_asr = textAsr;
```

约束：

- `text_asr` 与 NMT input 同源
- 删除 `text_asr_repaired`
- `extra.raw_asr_text` 继续来自 `ctx.rawAsrText`

# 6. 代码逻辑样例

## 6.1 ASR Step

```ts
function applyAsrResultToContext(ctx: PipelineContext, asrText: string) {
  const normalized = asrText.trim();

  if (!ctx.rawAsrText) {
    ctx.rawAsrText = normalized;
  }

  ctx.asrText = appendAsrDiagnosticText(ctx.asrText, normalized);

  if (!ctx.segmentForJobResult) {
    ctx.segmentForJobResult = normalized;
  }
}
```

---

## 6.2 FW Detector Apply

```ts
function applyFwDetectorResult(ctx: PipelineContext, detectorOutput: string) {
  const next = detectorOutput.trim();

  if (!next) {
    return;
  }

  ctx.segmentForJobResult = next;
  ctx.asrRepairApplied = true;
}
```

---

## 6.3 Segment Write Lock

```ts
function isSegmentWriteLocked(ctx: PipelineContext): boolean {
  return ctx.asrRepairApplied === true;
}
```

5015 / 5016 / 5017 约束：

```ts
if (isSegmentWriteLocked(ctx)) {
  return skipStep("segment write locked");
}
```

---

## 6.4 Aggregation

```ts
function runAggregation(ctx: PipelineContext, turnBuffer: TurnBuffer) {
  const segment = ctx.segmentForJobResult?.trim();

  if (!segment) {
    return;
  }

  if (!isFinalize(ctx)) {
    turnBuffer.committedSegments.push({
      jobId: ctx.jobId,
      text: segment,
      source: "segmentForJobResult"
    });

    ctx.shouldDeferTranslation = true;
    ctx.shouldAllowTranslation = false;
    return;
  }

  const fullTurnText = mergeSegments([
    ...turnBuffer.committedSegments.map(s => s.text),
    segment
  ]);

  ctx.segmentForJobResult = fullTurnText;
  ctx.shouldDeferTranslation = false;
  ctx.shouldAllowTranslation = true;
}
```

---

## 6.5 Translation

```ts
function runTranslation(ctx: PipelineContext) {
  if (ctx.shouldDeferTranslation) {
    return;
  }

  const input = getTextForTranslation(ctx);

  return translate({
    text: input.text,
    sourceLang: input.sourceLang,
    targetLang: input.targetLang
  });
}
```

---

## 6.6 Result Builder

```ts
function buildJobResult(ctx: PipelineContext): JobResult {
  return {
    job_id: ctx.jobId,
    text_asr: ctx.segmentForJobResult?.trim() || "",
    extra: {
      raw_asr_text: ctx.rawAsrText,
      translation_input_source: "segmentForJobResult",
      fw_detector_applied: ctx.asrRepairApplied === true
    }
  };
}
```
