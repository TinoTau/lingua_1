## 5. 建议的最简主链目标

```text
runJobPipeline(job):
  ctx = initJobContext(job)

  // ASR
  runAsrStep(job, ctx)
  ctx.rawAsrText      ← 首段 freeze（一次）
  ctx.asrText         ← 拼接观测（仅日志/extra）
  ctx.segmentForJobResult ← 首段业务初值

  // FW（zh/yue + feature on）
  runFwDetectorStep(job, ctx)
  detectInput  = ctx.rawAsrText
  applyOutput  → ctx.segmentForJobResult
  if approved: ctx.asrRepairApplied = true

  // Aggregation
  runAggregationStep(job, ctx)
  if turn && !finalize:
    appendTurnSegment(ctx.segmentForJobResult)
    deferTranslation = true
  if finalize:
    ctx.segmentForJobResult = mergeTurn(accumulator, segment)
    deferTranslation = false

  // 门控（无 segment 改写）
  applyPostAggregationRouting(...)

  // Dedup（只读 segment）
  if !defer: runDedupStep(segment)

  // Translation（只读 segment；空则 skip）
  if !defer && segment.nonEmpty:
    nmtInput = ctx.segmentForJobResult
    runTranslationStep(nmtInput)

  // Result
  text_asr = ctx.segmentForJobResult   // 无 asrText fallback
  extra.raw_asr_text = ctx.rawAsrText
```

**删除的控制流：** asrText→NMT fallback、currentSegment raw 回退、Recover steps、CTC extra 默认打包、双 flag 冗余表达。
