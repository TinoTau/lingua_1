## 🗂️ 文档分类说明

### 核心文档 (Core Documentation)
面向所有用户的技术文档，包括架构设计、API参考、配置说明等。这些是理解和使用服务的基础文档。

### 运维文档 (Operations Documentation)
面向运维人员的文档，包括部署、维护、故障排查和性能优化指南。

### 测试文档 (Testing Documentation)
面向测试人员和开发者的文档，包括测试指南、测试用例和测试结果总结。

### 脚本文档 (Scripts Documentation)
脚本工具的使用说明，包括日志查看、服务启动、测试运行等脚本的使用方法。

### 开发文档 (Development Documentation)
面向开发者的文档，包括日志规范、文件结构、服务注册等开发相关的技术文档。

### 总结报告 (Summaries)
各个阶段的总结报告，记录项目的里程碑和重要成果。

### 归档文档 (Archived Documentation)
历史文档，包括：
- **实现文档**: 记录实现过程的决策、计划和状态
- **问题报告**: 记录遇到的问题和解决方案
- **历史总结**: 各个阶段的总结报告

归档文档主要用于了解项目历史和决策背景，不影响日常使用。
